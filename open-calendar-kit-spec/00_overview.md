# OpenCalendarKit – Produktspezifikation

## Ziel
OpenCalendarKit ist ein kostenloses, international einsetzbares WordPress-Plugin für die Anzeige von Öffnungszeiten, Schließtagen, Monatskalendern und optionalen Status-/Hinweisblöcken.

Das Plugin entsteht als öffentliche, generische Weiterentwicklung aus einem bestehenden projektspezifischen Plugin. Das produktive Ursprungssystem bleibt getrennt und updatefähig. OpenCalendarKit wird als eigenes öffentliches Plugin entwickelt.

## Produktziel
Das Plugin soll für kleine bis mittlere Websites sofort nutzbar sein, ohne Reservierungslogik oder branchenspezifische Sonderfälle. Es soll:

- internationalisierbar und übersetzbar sein
- lokal korrekte Datums- und Zeitdarstellung unterstützen
- Unicode/UTF-8-tauglich sein
- mobil gut funktionieren
- als klassisches WordPress-Plugin leicht installierbar sein
- durch Shortcodes einfach in Seiten eingebunden werden können

## Public 1.0 Scope
In Scope:
- Öffnungszeiten pro Wochentag
- Monatskalender mit offenen/geschlossenen Tagen
- Schließtage / Ausnahmetage
- optionaler Status „heute geöffnet / geschlossen“
- optionale Event-/Hinweismitteilung
- Zusatzhinweis unter Öffnungszeiten
- Internationalisierung / Lokalisierung
- saubere Plugin-Dokumentation

Nicht in Scope für Public 1.0:
- Reservierungsanfragen
- Restaurant-spezifische Texte oder Workflows
- E-Mail-Versand
- mehrere Standorte / Mandanten
- ICS / API / externe Feeds
- Blocks statt Shortcodes
